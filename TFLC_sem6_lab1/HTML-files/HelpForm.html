О программе

Интерфейс программы

Область редактирования: Основное окно для ввода и редактирования текста. Поддерживает все стандартные операции (копирование, вставка, вырезание, отмена действий).

Область результатов: Окно для отображения результатов работы языкового процессора. (Ввод текста запрещен)

Особенности работы

Формат файлов

  • Поддерживается формат .txt
  • Кодировка UTF-8
  • Возможно расширение форматов

Навигация по ошибкам

  • Клик по сообщению об ошибке
  • Автоматический переход к позиции
  • Выделение ошибочного фрагмента

Дополнительные возможности

  • Подсветка синтаксиса
  • Строка состояния
  • Работа с несколькими файлами
